New Features

  • Ext 1.1.1
    Upgraded to Ext 1.1.1
  • GWT 1.4 Final
    Upgraded to GWT 1.4.60 (Final)
  • DelayedTask
    Utility class that provides a convenient method of performing setTimeout where a new timeout cancels the old timeout.
  • DialogListener
    Added DialogListener
  • Core Drag and Drop functionality
    Added core Drag and Drop classes under the com.gwtext.client.dd package
  • Editor and TreeEditor
    Added classes to support editing of Tree nodes
  • View and JsonView
    Added View and JsonView classes to support view based rendering
  • MultiSelectionModel
    Ability to multi-select tree nodes
  • TreeDragZone, TreeFilter and TreeSorter
    Support for Tree sorting, filtering and node editing
  • TreeEditor and TreeDropZone
    Better Tree Drag and Drop support
  • XMLTreeLoader and AsyncTreeNode
    Support for Asynchronous lazy loading of tree nodes. Also added XMLTreeLoader, a flexible asynchronous TreeDataLoader from XML data
  • Store
    Better exception handling. Users can register a listener for the onLoadException event.
  • Lot's more!
    A lot more new stuff has been added. See the API Diff docs

Breaking API Changes

This release also includes API changes that may require minor tweaks to existing code. Any such changes that affect you should only take a few minutes to rectify.

Grid Column Renderer

  • The render(..) function is now passed CellMetadata which provides the user a high degree of control on cell rendering.

*Field classes in the com.gwtext.client.data package

  • All *Field classes under the com.gwtext.client.data package have now bbeen renamed to *FeildDef. This was done in order to avoid confusion with similarly named classes under the com.gwtext.client.widgets.form package

Renderer

  • Renderer#render is now called with a value of type Object instead of String.

Breaking API Changes

This release also includes API changes that may require minor tweaks to existing code. Any such changes that affect you should only take a few minutes to rectify.

Grid, EditorGrid, BorderLayout

  • The constructors previously used to accept width and height in the order height, width but they now are in the order width, height.

EditorGridListener

  • The callback methods now have oldValue and newValue to be of type Object. Previously the arguments were of type String.

Renderer

  • Renderer#render is now called with a value of type Object instead of String.

Release Notes for 0.9

This is the first release of GWT-Ext. While a large percentage of the Ext API's are covered, currently there's not much in the form of user documentation, Javadocs and test cases. The idea is to put out a version of GWT-Ext for users to take a look at and provide feedback and report bugs. A large percentage of the Ext API's like data, grid, forms, layoutds, menus have very complete coverage in GWT-Ext. The most notable area that is missing is the Ext Drag and Drop (dd package) API's.

Please join the GWT-Ext Developer Forum to post questions and provide feedback.
