Class DataProxy

public class DataProxy
extends JsObject

// Superclass of ScriptTagProxy, MemoryProxy, HttpProxy
This class is an abstract base class for implementations which provide retrieval of unformatted data objects. DataProxy implementations are usually used in conjunction with an implementation of Reader (of the appropriate type which knows how to parse the data object) to provide a block of Record's to a Store.
